Genkey API

API for genkey analyzer made by @semilin. Get layout analysis from api as a json. Used Gin framework.

GET '/divided/'

Divide each row to another parameter. Query parameters:

  • topKeys = string of top row keys from left to right
  • homeKeys = string of home row keys from left to right
  • botKeys = string of bottom row keys from left to right
  • topFingers = string of top row fingers from left to right
  • homeFingers = string of home row fingers from left to right
  • botFingers = string of bottom row fingers from left to right
QWERTY example:
/divided/?topKeys=qwertyuiop&homeKeys=asdfghjkl%3B%27&botKeys=zxcvbnm,./&topFingers=0123344567&homeFingers=01233445677&botFingers=0123344567

GET '/together/'

All keys are included into one parameter. Query parameters:

  • keys = all keys from leftmost on top row to rightmost on bottom row
  • fingers = all fingers from leftmost on top row to rightmost on bottom row
QWERTY example:
/together/?keys=qwertyuiop[]\asdfghjkl%3B%27zxcvbnm,./&fingers=0123344567777012334456770123344567
